Personal web place

ujinf's web place.

여기는 포트폴리오보다 개인 웹사이트에 가깝습니다. 공개해도 되는 메모, 링크, 작은 실험, 다시 찾아볼 자료를 천천히 모아두는 장소입니다.

완성된 결과물만 정리하는 페이지가 아니라, 나중에 다시 열어볼 수 있는 작업 기록을 남기는 쪽을 우선합니다. 개인정보나 자세한 이력은 줄이고, 공개 가능한 생각과 자료의 맥락을 남깁니다.

Website

무엇을 담을까

이곳에는 완성된 자기소개보다 운영 중인 기록을 둡니다. 예를 들면 도메인 설정, GitHub Pages 배포, 작은 도구를 만들며 알게 된 점, 다시 찾아볼 링크, 프로젝트로 묶기에는 작은 실험 같은 것들입니다.

공개 범위는 의도적으로 좁게 잡습니다. 이름, 소속, 세부 일정처럼 굳이 첫 화면에 둘 필요가 없는 정보는 빼고, 사이트 방문자가 실제로 눌러볼 수 있는 자료와 글을 앞에 둡니다.
