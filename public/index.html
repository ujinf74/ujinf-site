project / C/C++ · Python · Unity

ballistic-solver

moving target을 gravity, quadratic drag, optional wind 조건에서 맞히기 위한 simulate-first intercept optimizer입니다. RK4로 비선형 projectile dynamics를 적분하고, closest-approach miss residual을 Levenberg-Marquardt로 줄입니다.

  • fixed-step RK4 with quadratic drag and wind as air velocity
  • damped least-squares, line-search/lambda tries, robust diagnostics
  • stable C ABI, PyPI package, Unity/C# P/Invoke integration
Repo PyPI
project / Isaac Sim · ROS 2 · Autoware

Mapless Autonomous Parking

prebuilt pointcloud/vector map 없이 online occupancy mapping에서 출발해 occupancy-only Hybrid A* replanning과 low-speed tracking을 연결한 mapless parking 작업입니다.

  • Autoware compatibility layer, self-cropping, ground filtering
  • ego-fixed local OGM, global accumulation, stale-cell decay
  • fraction-index interpolation and Stanley-based lateral control
project / MATLAB · telemetry · driver coaching

Racing Analyze GUI

여러 logged runs를 segment 기준으로 비교하고, driver/vehicle behavior를 시각화하는 racing telemetry analysis workbench입니다. real racing data의 visualization 및 coaching-oriented inspection workflow를 지원합니다.

  • GPS-fix-based track core and segment reference generation
  • XY, TS, MAP, POLAR, KDE, GG visualization modes
  • time-slider replay, pane pop-out, segment-level metrics extraction
